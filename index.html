NMAP - Network Mapper

Network exploration and security auditing tool

Installation:
sudo apt update sudo apt install nmap
(Kali Linux comes with nmap pre-installed)
Basic Scan:
nmap -sV -T4 <target_IP>
Aggressive Scan:
nmap -A -T4 <target_IP>
Port Range Scan:
nmap -p 1-1000 <target_IP>
Vulnerability Scan:
nmap --script=vuln <target_IP>
×

Metasploit Framework

Penetration testing platform

Installation:
sudo apt update sudo apt install metasploit-framework
(Comes pre-installed in Kali Linux)
Start Framework:
msfconsole
Search Exploits:
search exploit_name
Use Exploit:
use exploit/path set RHOSTS <target_IP> exploit
×

SQLMap

Automatic SQL injection tool

Installation:
sudo apt update sudo apt install sqlmap
(Pre-installed in Kali Linux)
Basic Scan:
sqlmap -u "http://site.com?id=1"
List Databases:
sqlmap -u "http://site.com?id=1" --dbs
Dump Data:
sqlmap -u "http://site.com?id=1" -D dbname -T users --dump
×

John the Ripper

Password cracking utility

Installation:
sudo apt update sudo apt install john
(Pre-installed in Kali as 'john')
Dictionary Attack:
john --wordlist=rockyou.txt hashes.txt
Incremental Mode:
john --incremental hashes.txt
Show Cracked Passwords:
john --show hashes.txt
×

Hydra

Parallelized login cracker

Installation:
sudo apt update sudo apt install hydra
(Pre-installed in Kali Linux)
SSH Brute Force:
hydra -l admin -P passwords.txt ssh://<target_IP>
FTP Attack:
hydra -L users.txt -p Password123 ftp://<target_IP>
×

Wireshark

Network protocol analyzer

Installation:
sudo apt update sudo apt install wireshark
(Pre-installed in Kali Linux)
Start Capture (CLI):
sudo wireshark
Common Filters:
http # HTTP traffic ip.src==192.168.1.1 # Traffic from IP tcp.port==80 # Port 80 traffic
